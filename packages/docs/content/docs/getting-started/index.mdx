# Getting Started

Welcome to the Streamflow JS SDK. This guide will walk you through setting up your environment and installing the packages you need.

## Prerequisites

- **Node.js >= 18** — the SDK is tested against Node 18, 20, 22, and 24.
- A Solana wallet or keypair for signing transactions.
- Basic familiarity with Solana programs and transactions.

## Next Steps

Head over to the [Installation](/docs/getting-started/installation) page to install the SDK packages and configure polyfills for your runtime environment.
